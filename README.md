# What's Behind the Door?

Jogo de plataforma 2D feito em **Godot 4.6** para a disciplina AC2 Games.

## Equipe (3º MIB — Trio)

| Integrante | Função |
|------------|--------|
| **Guilherme Pires** | Level design |
| **Leonardo de Paulo** | Personagem |
| **Matheus Rodrigues** | Documentação |

## Sobre o jogo

Lumi, um gato laranja perdido na escuridão, foge dos inimigos, coleta peixes e precisa alcançar a misteriosa porta no fim do mapa. Do outro lado: o **Reino dos Gatos**, um mundo iluminado e colorido.

## Controles

- **A / D** ou setas — mover
- **Espaço** — pular
- **Esc** — pausar

## Como rodar no Godot

1. Instale [Godot 4.6](https://godotengine.org/)
2. Abra o Godot → **Import** → selecione `project.godot`
3. Pressione **F5** para jogar

## Publicação (itch.io)

Instruções em `PUBLICAR_ITCH.md`. Exporte para **Web** e envie o zip gerado em `export/`.

## Estrutura principal

```
scenes/     # Cenas (menu, level, HUD, peixes, porta, reino dos gatos)
scripts/    # Scripts GDScript
assets/     # Ícones e sprites do HUD
```
